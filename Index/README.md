## 🎮 Sobre o Jogo

**Evolução Celular** é um jogo incremental onde você simula a origem da vida desde moléculas simples até o desenvolvimento do metabolismo da primeira célula.

Comece clicando em **Aminoácidos**, **Nucleotídeos** e **Lipídios** - os blocos fundamentais da vida. Combine-os para formar moléculas orgânicas, evolua para RNA, forme membranas, sintetize DNA e finalmente desenvolva o **Metabolismo**!

## ✨ Características

- 🧬 **6 estágios evolutivos** (Moléculas → RNA → Membrana → DNA → Metabolismo)
- 📈 **6 sistemas de upgrade** (Expansão, Automações, Clonagem)
- 🏆 **8 conquistas** desbloqueáveis com notificações
- 💾 **Save/Load** em arquivo .txt (compartilhável!)
- 🎨 **Interface neon** com tema científico
- ⚡ **Automações progressivas** (de 60s até 10s!)
- 🔄 **Sistema de clonagem** com chance de DNA extra

## 🎯 Como Jogar

1. **Clique nos 3 botões básicos** para acumular recursos
2. **Gere moléculas orgânicas** (10 de cada recurso)
3. **Sintetize RNA** (20 moléculas)
4. **Forme Membranas** (10 RNA)
5. **Crie DNA** (4 membranas)
6. **Desenvolva o Metabolismo** (100 DNA) - **VITÓRIA!**

### Dicas
- Invista em **upgrades de capacidade** cedo
- **Automações** são essenciais para progresso rápido
- **Clonagem** pode dobrar sua produção de DNA
- Use o **menu lateral** para salvar seu progresso
